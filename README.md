### Build instructions
Publish to expo
- npm run expo-publish

Build for iOS
- npm run build-ios

Build for Android
- npm run android

Run local preview
- npm start
- then enter the local url into iphone's browser with exp:// prefix
- or scan the qr code using any qr code scanner
