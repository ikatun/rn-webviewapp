### Build instructions
`npm install` to install all dependencies, then:

Publish to expo
- `npm run expo-publish`

Build for iOS
- `npm run build:ios`

Build for Android
- `npm run build:android`

Run local preview
- npm start
- then enter the local url into iphone's browser with exp:// prefix
- or scan the qr code using any qr code scanner
